🏥 Clinic Website

A modern and responsive clinic website designed to enhance patient experience through easy navigation, service visibility, and seamless appointment booking.

🚀 Features

🩺 Hero Section with Credentials
Displays doctor qualifications (M.B.B.S, M.D., Certified Nutritionist)

👨‍⚕️ Doctor Profile Section
Highlights experience, specialization, and qualifications

💊 Treatment Services
9 treatment cards with tags and “Book Now” options

⭐ Why Choose Us
Key highlights with statistics for trust-building

🖼️ Photo Gallery
Clinic and dermatology-related images

💬 Patient Testimonials
Real reviews to build credibility

📅 Appointment Booking Form
Easy form with treatment selection dropdown

📱 WhatsApp Integration
Direct contact for quick communication

📍 Google Maps Integration
Displays clinic location for easy navigation

📲 Fully Responsive Design
Optimized for mobile, tablet, and desktop

🎯 UI Enhancements
Sticky navbar and smooth scroll animations
